## 🔐 Ciberseguridad

### ⚡ Evil Twin Attack — Simulación de Punto de Acceso Fraudulento

Proyecto práctico sobre uno de los ataques WiFi más peligrosos del mundo real. Simulación de un AP malicioso que suplanta una red legítima para interceptar tráfico de usuarios conectados.

`Python` `Kali Linux` `hostapd` `dnsmasq` `Wireless Networking`

**Áreas cubiertas:** configuración de AP falso · captura de credenciales · análisis de tráfico · contramedidas defensivas

---

### 🔓 SSL Strip — Investigación sobre Ataques de Degradación HTTPS

Investigación y demostración del ataque SSL Stripping, que fuerza conexiones HTTPS a HTTP para interceptar comunicaciones cifradas en texto plano.

`Python` `Kali Linux` `Wireshark` `MITM` `Network Security`

**Áreas cubiertas:** ataques Man-in-the-Middle · degradación de protocolo · análisis con Wireshark · mitigaciones (HSTS, cert pinning)
